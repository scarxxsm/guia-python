# Input y Output (Bases)

Es esencial conocer lo que significan estas palabras en la programación.
Cuando hablamos de un **input**, nos referimos a los datos que el programa **recibe**.
Por el contrario, cuando hablamos de un **output**, nos indica las salidas que tendrá este programa.

- Output:
Para realizar un output en python buscamos una salida, en este caso usamos la *terminal* para mostrar todos los mensajes que queramos, en codigo escribimos: `print("x")`
Sustituimos la x por el mensaje que queramos, ej. `print("Hola mundo")`
En las lecciones siguientes aprenderemos sobre variables, pero es importante mencionar que también podemos usar `print()` con variables, ej. 
```python
nombre = "Alexis"
print(nombre)
```

- Input:
Hacemos un input cuando nos interesa que el usuario ingrese información. Usamos `input("x")`
Es importante que asignemos ese valor a una variable.
Ejemplo, si queremos preguntar el nombre del usuario escribimos algo así: 
```python
nombre = input("Dime tu nombre: ")
```
Si queremos conocer la edad o valores que sean **numéricos**, usamos **int**.
Ejemplo, si queremos conocer la edad del usuario escribimos algo así:
```python
edad = int(input("Dime tu edad: "))
```
