## Variables y Tipado Dinámico

En otros lenguajes como lo es Java, cuando declaramos una variable se debe especificar su tipo.
Por ejemplo:
```java
String nombre = "Juan";
int edad = 24;
double precio = 3.57;
```
Sin embargo, Python usa *Tipado Dinámico:* el lenguaje detecta el tipo de variable según el valor
que le asignes.

### Tipos Primitivos en Python

| Tipo | Nombre Técnico | Ejemplo de Código en Python | Nota |
| ----------- | ----------- | ----------- | ----------- |
| Entero | int | `edad = 25` | No tiene límite de tamaño. |
| Decimal | float | `precio = 19.99` | Maneja altaprecisión decimal. |
| Texto | str | `nombre = "Alexis"` | Se deben usar comillas simples ' ' o dobles " " |
| Booleano | bool | `activo = True` | Deben empezar obligatoriamente con mayúscula: True o False. |

## Operadores

### Aritmeticos:
Estos sirven para realizar operaciones matemáticas.

| Tipo | Sintaxis | Ejemplo | Output |
| :----------- | :-----------: | :-----------: | :-----------: |
| Suma | + | `4 + 5` | 9 |
| Resta | - | `9 - 3` | 6 |
| Multiplicación | * | `5 * 3` | 15 |
| División | / | `20/4` | 5 |
| División entera | // | `7//2` | 3 |
| Exponente/Potencia | ** | `4**2` | 16 |
| Módulo(residuo) | % | `7%2` | 1 |

### De Comparación y Lógicos:
Estos se usan normalmente en sentencias condicionales y siempre devuelven un booleano (True o False).

| Tipo | Sintaxis | Ejemplo | Output |
| :----------- | :-----------: | :-----------: | :-----------: |
| Igual | == | `4 == 4` | True |
| Diferente | != | `4 != 6` | True |
| Mayor | > | `9 > 2` | True |
| Menor | < | `3 < 8` | True |
| Y | and | `True and True` | True |
| No | not | `not False` | True |
| O | or | `True or False` | True |

## Control de flujo

Python no usa llaves *{}* para crear bloques de código.
En su lugar usa la **identación** (4 espacios o un tabulador).
También se usan los dos puntos (**:**) para indicar el inicio de cada bloque.
