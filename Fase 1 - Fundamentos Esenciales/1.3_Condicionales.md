## Condicionales: if, elif, else

Los condicionales son metodos que nos permiten, como dice su nombre, saber cuando una confición se cumple. Aunque podemos pensarlos como preguntas de 'sí' y 'no', Python no se limita solo a las respuestas True y False, sino que evalúa la 'verdad' o 'falsedad' de cualquier objeto.

> En este tema trabajaremos mucho con los **operadores**, si no los recuerdas puedes revisar en archivo *1.2_Variables_Operadores*

### if:

**if** se traduce directamente al español como "si", ejemplo:

- **if** acabaste tu tarea, **entonces** sal a jugar
- **if** tienes sueño, **entonces** duerme

*Sintaxis:*

```python
if condicion:
    instruccion_unica
```
Es importante recordar el *control del flujo* en python. Todo lo que contiene el **if**, debe de estar dentro de un bloque de código, para crear este **"bloque"** ponemos **":"** al final de la linea (como en el ejemplo), y la siguiente línea debe comenzar con un tabulador (cuatro espacios o un tab).

*Ejemplo:*
```python

edad = 20
if edad >= 18:
    print("Admitido, eres mayor de edad")
```
Este código nos arrojaría este mensaje: *Admitido, eres mayor de edad*, ya que la edad que declaramos fue 20, como 20 es mayor que 18 la condición se *cumple*.

---

En muchos casos es necesario utilizar mas de un condicional, por ejemplo en en cuestionario:
Supongamos que queremos automatizar el acceso a un club, dependiendo la edad del usuario, se le otorgará una etiqueta distinta
1) 18 - 25 --> verde
2) 26 - 32 --> roja
3) +33 --> amarilla
4) -18 --> Sin acceso 

Lo bueno de la progrmación es que existen muchas variantes para resolver los problemas, en esta ocasión veremos las más básicas.

Opción 1:
```python

if 18 <= edad <= 25:
    print("verde")
if 26 <= edad <= 32:
    print("roja")
if edad >= 33:
    print("amarilla")
if edad < 18:
    print("Sin acceso")

```
Este código funciona pero hay más formas de hacerlo, por ejmplo, usando **elif** -->
## elif
**elif** nos es útil cuando solo necesitamos que *una* de *todas* las condiciones se cumpla. **elif** siempre va después de un **if**.
Por ejemplo si necesitamos verificar varios parametros en los que devolveremos un distinto valor usamos **if**, de esta manera nos aseguramos de que cada condición sea leída. Cuando usamos **elif**, el programa dejará de leer las demás condiciones cuando encuentre una que se cumpla. 
Volviendo a nuestro ejemplo anterior quedaría de está manera:

Opción 2:
```python

if 18 <= edad <= 25:
    print("verde")
elif 26 <= edad <= 32:
    print("roja")
elif edad >= 33:
    print("amarilla")
elif edad < 18:
    print("Sin acceso")

```

Anteriormente al usar unicamente **if** no importaba si el usuario cumplia la primera condición, *18=< edad >= 25*, el programa seguía leyendo las demás condiciones en busca de mas coincidencias. 
En cambio, ahora que usamos **elif** si el usuario cumple la primera condición, *18=< edad >= 25*, el programa dejará de leer las demás coincidencias porque sabe que no se cumplirán o que no es necesario que se cumplan.

**elif** es útil cuqando tenemos que verificar que unicamente una condición se cumpla.
