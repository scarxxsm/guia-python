# Mi guía de Python desde cero
Una guía práctica y desde cero de los fundamentos de Python, totalmente en español.  

---

## ¿Por qué Python y para quién es esto?
* Python es uno de los lenguajes más limpios y fáciles de aprender  
* Es ideal como tu primer lenguaje, ya que la sintaxis es como dar  
indicaciones en inglés.  

---

## Temario:
* Fase 0: Introducción al lenguaje y repositorio de herramientas de aprendizaje y práctica  
* Fase 1: Fundamentos Esenciales  
* Fase 2: Estructuras de Datos y Manipulación de Datos  
* Fase 3: Modularidad, Funciones y Control de Errores  
* Fase 4: Programación Orientada a Objetos (POO)  
* Fase 5: Python Avanzado  
* Fase 6: El Entorno Profesional y Buenas Prácticas  
* Fase 7: Conexión con el Mundo Real (Aplicaciones Prácticas)  

## Modo de trabajo:

- Cada fase contará con su propia carpeta, ahí estarán documentos de teoría ordenados para su mejor comprensión.
- Estás carpetas contarán con ejercicios básicos para que practiques y estes seguro de haber entendido el tema.
- Al final de cada fase encontrarás un proyecto final (la dificultad varia dependiendo la fase en la que estes).
Te invito a usar **Github** para subir tu solución, si no estás seguro de que sea la solución correcta, 
puedes checar la solución dada.
